# Versiones anteriores del desarrollo

Esta carpeta conserva versiones previas del proyecto tal como estaban organizadas en la PC, separadas del flujo final de `CARACTERIZACION_FINAL`.

- `CODEX_CARACTERIZACIOIN`: version anterior de caracterizacion EMG. Incluye scripts numerados de adquisicion, entrenamiento, simulacion e inferencia en vivo, ademas de datasets historicos.
- `DESARROLLO_control_caracterizado`: firmware anterior de control caracterizado.

No se versionan entornos virtuales, instaladores de Python, caches ni modelos Random Forest gigantes (`emg_sklearn_baseline.joblib`) porque superan el limite practico de GitHub y pueden regenerarse con los scripts de entrenamiento.
